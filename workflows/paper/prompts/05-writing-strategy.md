# Stage 5: Writing Strategy
# Develop the writing strategy based on reviewer critique.

## Inputs
- `critical-review.yaml`
- `paper-state.yaml`
- `claim-evidence-grounding.yaml`

## Task
Produce `writing-strategy.yaml`:

1. **Narrative strategy**: What's the paper's one-sentence story?
   Problem → Root Cause → Insight → Design → Evaluation → Contribution.

2. **Section strategy** per target unit:
   - What must this section accomplish for the reader?
   - What's the rhetorical pattern? (problem-root-cause-insight, claim-evidence, etc.)
   - What claims must this section support?

3. **Claim strategy**:
   - Which claims to strengthen? How?
   - Which claims to weaken? New wording?
   - Which claims to drop?

4. **Writing tactics**:
   - SOSP/OSDI conventions: figure-first, claim-up-front, no throat-clearing
   - Transitions between sections
   - Signposting for reviewer navigation
