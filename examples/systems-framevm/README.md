# FrameVM — Systems Paper Example

This directory contains FrameVM-specific examples that illustrate the CriticalResearch workflow applied to a systems paper. These are examples, not templates. The generic workflow, roles, and templates in `workflow/`, `roles/`, and `templates/` are domain-agnostic and should be used for all projects.

## Contents

- Example paper patches for isolation/virtualization claims
- Example evaluation obligations for systems benchmarks
- Example knowledge cards derived from OS/systems research
